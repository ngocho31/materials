\subchapter{Fetching Bootloader - U-Boot sources}{Objectives:
fetch the U-Boot sources from git.}

\section{Cloning the U-Boot sources}

Download U-Boot:

\begin{bashinput}
$ git clone https://source.denx.de/u-boot/u-boot.git/
$ cd u-boot
\end{bashinput}

We will use the U-Boot \workinguboot.

\begin{bashinput}
$ git checkout v%\workinguboot%
\end{bashinput}
